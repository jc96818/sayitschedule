## When to use this

- You need to add a new {{labels.staff.singular}} to your organization.
- You want to view, search, or filter your {{labels.staff.plural}} list.
- You need to activate or deactivate a {{labels.staff.singular}}.
- You want to delete a {{labels.staff.singular}} from the system.

## How it works

The {{labels.staff.plural}} page displays all {{labels.staff.plural}} in your organization in a searchable, filterable table. You can add new {{labels.staff.plural}} using either voice commands or a form. Each {{labels.staff.singular}} has a profile page where you can edit their details, set working hours, and manage time off requests.

## Steps

### View your {{labels.staff.plural}}

1. Open **{{labels.staff.plural}}** from the sidebar.
2. The table displays all {{labels.staff.plural}} with their name, gender, email, phone, {{labels.certification.plural}}, and status.
3. Use the search box to find {{labels.staff.plural}} by name or email.
4. Use the **Status** dropdown to filter by Active or Inactive.
5. Use the **Gender** dropdown to filter by Male or Female.

### Add a {{labels.staff.singular}} using voice

1. On the {{labels.staff.plural}} page, click the microphone button or select **Speak**.
2. Say a command like "Add a new {{labels.staff.singular}} named Sarah Johnson, female" or "New {{labels.staff.singular}} Adam Smith, male, certified in ABA therapy".
3. Review the AI interpretation that appears.
4. Click **Add {{labels.staff.singular}}** to confirm, or click **Edit Details** to modify the information before saving.
5. Click **Cancel** to discard and try again.

### Add a {{labels.staff.singular}} using the form

1. Click **Add {{labels.staff.singular}}**.
2. Enter the following information:
   - **Full Name** (required)
   - **Email** (required)
   - **Phone** (optional)
   - **Gender** – select Male or Female
   - **Max Sessions Per Day** – the maximum number of sessions this {{labels.staff.singular}} can have in one day (default is 2)
3. Click **Add {{labels.staff.singular}}** to save.

### Edit a {{labels.staff.singular}}

1. From the {{labels.staff.plural}} list, click **View** next to the {{labels.staff.singular}} you want to edit.
2. On the profile page, click **Edit Profile**.
3. Update the fields as needed: Full Name, Gender, Email, Phone, or Hire Date.
4. Click **Save Changes**.

### Activate or deactivate a {{labels.staff.singular}}

1. Open the {{labels.staff.singular}}'s profile page.
2. In the **Personal Information** section, toggle the **Status** switch.
3. Active {{labels.staff.plural}} are available for scheduling; inactive {{labels.staff.plural}} are not.

### Delete a {{labels.staff.singular}}

1. Open the {{labels.staff.singular}}'s profile page.
2. Click **Delete**.
3. Confirm the deletion when prompted. This action cannot be undone.

## How this changes with your settings

### Voice input vs. typed commands

You can add {{labels.staff.plural}} by speaking naturally or by typing a command. The voice interface includes **Speak** and **Type** modes. Example commands:

- "Add a new {{labels.staff.singular}} named Sarah Johnson, female"
- "New {{labels.staff.singular}} Adam Smith, male, certified in ABA therapy"
- "Add {{labels.staff.singular}} Maria Garcia with speech therapy and pediatrics certifications"

If microphone access is blocked or voice transcription isn’t configured, use **Type** mode or the **Add {{labels.staff.singular}}** form.

## Troubleshooting

- **"No {{labels.staff.plural}} found"**: Clear search and filters (Status/Gender) to see the full list.
- **Voice command not understood**: Try a shorter command, or use **Edit Details** to correct the parsed values before saving.
- **Microphone not working**: Allow microphone access for this site, or switch to **Type** mode.
- **Cannot add {{labels.staff.singular}}**: Only Admins, Admin Assistants, and Super Admins can manage {{labels.staff.plural}}.
